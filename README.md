# 数据收集工作台

## 软件用途

本软件用于临床科研或病例资料整理过程中的**人工数据收集**。  
适用于院内单机、无联网环境下，基于 Excel 表格进行逐病例、逐字段录入的场景。

---

## 软件功能

- 导入 Excel 表格并生成录入界面
- 以“住院号”为索引逐个病例录入
- 点击住院号可快速复制
- 支持首页、上一页、下一页、末页、跳转
- 已填写与未填写字段颜色区分
- 支持界面设置与滚动显示
- 自动保存录入内容

---

## Excel 表格要求

- 第一行为列名
- 必须包含一列命名为 **“住院号”**
- 其余列名为需要收集的变量名称

---

## 软件优点

- 适合院内单机流数据收集
- 无需来回切换多个窗口
- 可在单台设备上完成录入
- 简单直接，便于上手

---

## 软件缺点

- 界面较为简陋
- 仍依赖人工逐项录入
- 由于使用 Python 打包，体积约 **25–30 MB**

---
